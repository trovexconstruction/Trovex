موقع Trovex — نسخة مجانية جاهزة للنشر

الملفات الرئيسية:
- index.html
- styles.css
- script.js
- assets/logo.svg

طريقة التشغيل:
افتح ملف index.html في أي متصفح.

طريقة النشر المجاني على GitHub Pages:
1) أنشئ حسابًا على GitHub.
2) أنشئ مستودعًا جديدًا باسم trovex-website.
3) ارفع محتويات هذا المجلد.
4) من Settings > Pages اختر Deploy from a branch ثم main / root.
5) سيظهر الموقع على رابط مجاني تابع لـ GitHub.

ملاحظات:
- نموذج التواصل يفتح واتساب مباشرة برسالة جاهزة.
- الموقع متجاوب مع الهاتف والكمبيوتر.
- يمكن استبدال النصوص والمشروعات بسهولة من ملف index.html.
